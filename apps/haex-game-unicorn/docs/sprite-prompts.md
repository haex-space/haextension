# Sprite Generation Prompts for ChatGPT / Gemini

## Style-Anweisung (einmal am Anfang der Session geben)

I need pixel art sprites for a 2D top-down nature game for preschool children. The style should be: hi-bit pixel art (like modern indie games such as Eastbound or Stardew Valley), warm and colorful, child-friendly, detailed but readable. All sprites must have a transparent background (PNG). The perspective is top-down with slight 3/4 view (like classic SNES RPGs). Keep the style consistent across all images I request in this session.

---

## 1a. Einhorn Spritesheet (HOECHSTE PRIORITAET)

Create a pixel art spritesheet of a cute unicorn character for a top-down 2D game. The unicorn should be white/light pink with a golden horn and a flowing pastel rainbow mane/tail. Style: hi-bit pixel art, 3/4 top-down view.

The spritesheet should show the unicorn in a grid layout:
- Row 1: Walking DOWN (facing camera) - 4 frames
- Row 2: Walking LEFT - 4 frames
- Row 3: Walking RIGHT - 4 frames
- Row 4: Walking UP (facing away) - 4 frames

Each frame should be 64x64 pixels. The unicorn should look cute and friendly, suitable for preschool children. Transparent background.

---

## 1b. Einhorn Idle

Create a pixel art sprite of the same cute unicorn character, standing still in an idle pose, facing down-right (3/4 view). The unicorn gently sways or breathes. 64x64 pixels, hi-bit pixel art, transparent background. Show 2 frames side by side (idle animation).

---

## 2a. Hummelkoenigin Spritesheet (HOECHSTE PRIORITAET)

Create a pixel art spritesheet of a bumblebee queen for a top-down 2D nature game. The bumblebee should be anatomically recognizable: plump fuzzy body with black and golden-yellow stripes, translucent wings, visible legs, and antennae. She should look cute but realistic - not cartoonish. Hi-bit pixel art, 3/4 top-down view.

The spritesheet should show:
- Row 1: Flying RIGHT - 4 frames (wings flapping)
- Row 2: Flying LEFT - 4 frames
- Row 3: Sitting/resting - 2 frames
- Row 4: Collecting nectar from a flower - 2 frames (head down into flower)

Each frame 48x48 pixels. Transparent background. The pollen baskets on the hind legs should be visible (empty in these frames).

---

## 2b. Hummel mit Pollen

Create the same bumblebee queen as before, but now with visible orange pollen balls attached to her hind legs (pollen baskets). Show her flying right with pollen - 4 frames, 48x48 each. Hi-bit pixel art, transparent background. The pollen should be clearly visible as bright orange clumps on the legs.

---

## 3a. Gras-Tileset Fruehling

Create a pixel art tileset of spring meadow grass for a top-down 2D game. Hi-bit pixel art style. The tileset should contain 9 tiles in a 3x3 grid:

- 4 basic grass variations (slightly different shades of fresh spring green)
- 2 grass with small wildflowers (tiny white/yellow dots)
- 1 grass with a small dirt patch
- 1 grass with tiny mushrooms
- 1 tall grass tuft

Each tile 32x32 pixels. The tiles should seamlessly tile together. Transparent background. Colors: fresh spring green, light green, hints of yellow-green.

---

## 3b. Gras-Tileset Sommer/Herbst/Winter

Create 3 variations of the same grass tileset (same 3x3 grid layout), but for different seasons:

1. Summer: Deep rich green, more flowers, lusher
2. Autumn: Yellow-orange-brown tones, some fallen leaves on the ground
3. Winter: Muted grey-green with snow patches, frost on blades

Same layout as the spring version, 32x32 per tile, pixel art, transparent background.

---

## 4. Baeume pro Jahreszeit

Create 4 pixel art trees for a top-down 2D nature game, one for each season, arranged side by side:

1. Spring tree: Light green leaves, some pink/white blossoms, thin brown trunk
2. Summer tree: Full dark green canopy, lush and round, thick trunk
3. Autumn tree: Orange/red/golden leaves, some falling
4. Winter tree: Bare branches with snow, no leaves, maybe a few remaining brown leaves

Each tree approximately 64x96 pixels (tall). 3/4 top-down perspective. Hi-bit pixel art. Transparent background. The trees should cast a small shadow to the bottom-right.
